# Treeline Releases

This repository hosts release assets for the [Treeline](https://treeline.money) desktop app.

## About

Treeline is a local-first personal finance app. The source code is private, but release binaries are published here to enable automatic updates.

## Downloads

See the [Releases](https://github.com/treeline-money/treeline-releases/releases) page to download the latest version.

## Auto-Updates

The Treeline desktop app checks this repository for updates automatically. When a new version is available, you'll be prompted to install it.
